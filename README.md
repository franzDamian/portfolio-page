# Franz Schulze - Portfolio

Ein moderner, professioneller Portfolio-Website für einen Full-Stack Software-Entwickler.

## 🚀 Schnellstart

### Option 1: Direkt auf GitHub Pages deployen

1. **Repository erstellen**
   - Gehe zu [github.com](https://github.com)
   - Erstelle ein neues Repository: `username.github.io`
   - Clone es auf deinen Computer

2. **Dateien kopieren**
   - Kopiere `index.html`, `style.css` und `script.js` in das Repository

3. **Deploy**
   - Commit und push die Änderungen
   - Deine Seite ist unter `https://username.github.io` erreichbar

### Option 2: Bestehendes Repository nutzen

Falls du dieses Repository bereits als GitHub Pages aktivieren möchtest:

1. Gehe zu **Settings** → **Pages**
2. Wähle unter **Source** den `main` Branch
3. Speichere die Einstellungen

## 📁 Dateistruktur

```
portfolio-page/
├── index.html    # Hauptseite mit allen Inhalten
├── style.css     # Styling (Dark Theme, Modern Design)
├── script.js     # Interaktive Elemente
└── README.md     # Diese Datei
```

## 🎨 Design-Features

- **Dark Theme** mit moderner Ästhetik
- **Responsive Design** für alle Geräte
- **Animierte Code-Visualisierung** im Hero-Bereich
- **Smooth Scrolling** Navigation
- **Interaktive Skill-Tags**
- **Timeline** für Berufserfahrung
- **Kontakt-Bereich** mit Links

## 🔧 Anpassung

### Kontaktdaten ändern

In `index.html` die folgenden Stellen anpassen:

```html
<!-- Zeile ~200: E-Mail -->
<a href="mailto:fda.schulze@gmail.com">
	<!-- Zeile ~210: GitHub Link -->
	<a href="https://github.com/DEIN-USERNAME">
		<!-- Zeile ~213: LinkedIn Link -->
		<a href="https://linkedin.com/in/DEIN-USERNAME"></a></a
></a>
```

### Farben anpassen

In `style.css` die CSS-Variablen ändern:

```css
:root {
	--accent-primary: #6366f1; /* Primärfarbe ändern */
	--accent-secondary: #8b5cf6; /* Sekundärfarbe ändern */
}
```
